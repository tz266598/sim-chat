## 项目概述

### 1.1 项目背景

SIM-Chat 是为武汉大学信息管理学院开发的智能问答系统，旨在帮助学生快速获取学院相关信息，包括培养方案、课程设置、师资介绍、通知公告等。

### 1.2 核心功能

- ✅ **智能问答**: 基于自然语言的理解和回答
- ✅ **文档检索**: 从学院文档中检索最相关内容
- ✅ **来源追溯**: 回答附带参考文档来源
- ✅ **对话界面**: 简洁的学术风格Web界面

### 1.3 技术特点

- **RAG 架构**: 检索增强生成（Retrieval-Augmented Generation）
- **模块化设计**: 前后端分离，易于维护和扩展
- **轻量部署**: 支持Docker一键部署或本地运行

### 3.2 数据流

```
用户提问
  │
  ▼
[1] 问题向量化 (MiniMax Embedding API)
  │ 输入: 自然语言问题
  │ 输出: 1536维向量
  │
  ▼
[2] 文档检索 (关键词匹配 / 向量相似度)
  │ 输入: 问题向量
  │ 输出: Top-K 相关文档
  │
  ▼
[3] 上下文构建
  │ 输入: 检索到的文档
  │ 输出: 格式化的上下文字符串
  │
  ▼
[4] 回答生成 (MiniMax LLM API)
  │ 输入: 问题 + 上下文
  │ 输出: 自然语言回答
  │
  ▼
[5] 返回结果
  │ 包含: 回答 + 来源列表
  │
  ▼
用户接收回答
```

#### 4.1.2 MiniMax API封装 (`minimax.py`)

```python
class MiniMaxClient:
    def __init__(self):
        self.api_key = settings.MINIMAX_API_KEY
        self.base_url = "https://api.minimax.chat/v1"
    
    def get_embedding(self, text: str) -> list:
        """文本向量化"""
        url = f"{self.base_url}/embeddings"
        payload = {
            "model": "embo-01",
            "texts": [text],
            "type": "db"
        }
        response = httpx.post(url, json=payload, headers=headers)
        return response.json()["vectors"][0]
    
    def get_completion(self, prompt: str, context: str) -> str:
        """LLM文本生成"""
        url = f"{self.base_url}/text/chatcompletion_v2"
        payload = {
            "model": "MiniMax-M1",
            "messages": [
                {"role": "system", "content": system_prompt},
                {"role": "user", "content": prompt}
            ]
        }
        response = httpx.post(url, json=payload, headers=headers)
        return response.json()["choices"][0]["message"]["content"]
```

**设计要点**:
- 统一的API接口管理
- 错误处理和重试机制
- 支持多种API响应格式

#### 4.1.3 向量数据库 (`vectorstore.py`)

```python
class VectorStore:
    def __init__(self):
        # 使用本地持久化数据库
        db_path = Path(__file__).parent.parent.parent.parent / "data_processing" / "chroma_db"
        self.client = chromadb.PersistentClient(path=str(db_path))
        self.collection = self.client.get_collection(name="sim_chat_docs")
    
    def query(self, query_embedding: list, n_results: int = 5) -> dict:
        """向量相似度查询"""
        return self.collection.query(
            query_embeddings=[query_embedding],
            n_results=n_results,
            include=["documents", "metadatas", "distances"]
        )
```

**设计要点**:
- 本地持久化存储
- 支持向量检索和关键词检索
- 自动降级机制

#### 4.1.4 检索服务 (`retrieval.py`)

```python
def retrieve_documents(query: str, top_k: int = 5) -> list:
    """检索相关文档"""
    try:
        # 尝试向量检索
        query_embedding = minimax_client.get_embedding(query)
        results = vector_store.query(query_embedding, n_results=top_k)
        return format_results(results)
    except Exception as e:
        # 降级到关键词检索
        return keyword_search(query, top_k)

def keyword_search(query: str, top_k: int = 5) -> list:
    """关键词匹配检索"""
    results = vector_store.collection.get(include=["documents", "metadatas"])
    # 基于关键词匹配度排序
    return sorted_results[:top_k]
```

**设计要点**:
- 自动降级机制（向量检索失败时回退到关键词检索）
- 支持多种检索策略
- 结果格式统一
